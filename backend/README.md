# full-stack-workshop
ACM workshop where a back-end and a front-end application will be made, join both, and culminate in an Full Stack project.

## Development
`docker-compose up --build` should do everything you need.

## Backend
Technologies used:
- Nodejs
- Passportjs
- MongoDB
- JWT (JSON Web Tokens)

## environments
- SECRET_KEY
- DB_HOST
- CLIENT_SERVICE
- EMAIL_SERVICE
- EMAIL_ADDRESS
- EMAIL_PWD
- EMAIL_SMTP_USER
- EMAIL_SMTP_PWD
- PORT
